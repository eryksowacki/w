Koło zębate + Zarządzaj skrótami rozszerzeń
ChatGPT Moodle Helper

Skopiuj prompt z aktualnym pytaniem       → Alt + Shift + C
Otwórz ChatGPT z aktualnym pytaniem       → Alt + Shift + Q
Skopiuj prompt ze wszystkimi pytaniami    → Alt + Shift + A
Otwórz ChatGPT ze wszystkimi pytaniami    → Alt + Shift + O

Jeżeli któryś skrót nie działa, użyj zamienników, np.:

Ctrl + Shift + Y
Ctrl + Shift + U
Ctrl + Shift + I
Ctrl + Shift + O

Jedno pytanie:

Kliknij lewym przyciskiem w treść pytania albo jedną z odpowiedzi.
Naciśnij:
Alt + Shift + C
Powinien pojawić się mały komunikat:
Prompt skopiowany
Przełącz się do ChatGPT:
Alt + Tab
Wklej:
Ctrl + V
Wyślij:
Enter



Otwieranie ChatGPT automatycznie

Dla jednego pytania:

Alt + Shift + Q

Dla wszystkich pytań ze strony:

Alt + Shift + O

Uwaga: przy wielu pytaniach lepiej używać Alt + Shift + A, bo URL dla ChatGPT może być za długi.

Najpewniejszy wariant:

Alt + Shift + A → Alt + Tab → Ctrl + V → Enter
